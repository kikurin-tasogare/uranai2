# 占蔵 -SENZO- 世界の占い大図鑑

## コンセプト
世界中の占いを「**知って、試せる**」大図鑑ウェブサイト。
各占術は「知る(図鑑:由来・歴史・仕組み)」と「占う(本格計算による鑑定)」の二つの顔を持つ。
「こんな占いもあるんだ」という発見体験と、占術ロジックへの忠実さ(精度)が二本柱。

- まずは自分と祐希ちゃんの2人で使う。将来は一般公開も視野(図鑑ページがそのままSEO資産になる)
- プラットフォームはウェブ(スマホ対応)。人気が出たらPWA/アプリ化を検討
- 鑑定文は固定辞書でスタート。AI統合鑑定は将来のオプション

## 収蔵済みの占術
| 占術 | 系譜 | 計算方式 |
|---|---|---|
| 宿曜占星術 | インド→唐→日本(空海将来) | 旧暦(天保暦置閏法・JST)を天文計算し、宿曜道伝統の暦法で二十七宿を算出。三九の秘法で今日の運気と相性(六種の縁)も |
| 四柱推命 | 中国 | 節入り実時刻(天文計算)で年柱・月柱、JDNで日柱、真太陽時(経度+均時差)で時柱。通変星・五行バランス・大運・年運付き |
| 西洋占星術 | バビロニア→ギリシャ→西欧 | astronomy-engineで10天体の視黄経+アセンダント(トロピカル方式)。主要アスペクト・エレメントバランス付き |
| マヤ暦(ツォルキン) | マヤ文明(独立系) | GMT対応(584283)の伝統マヤ。20紋章×13音、ロングカウント・ハアブ暦・カレンダーラウンド(52年)も |
| 易経 | 中国(殷周) | 擲銭法(老陰6/少陽7/少陰8/老陽9)を忠実に再現。六十四卦・変爻・之卦 |
| 紫微斗数 | 中国(宋代)→台湾・香港 | 旧暦+時辰(真太陽時)から命宮・身宮・納音五行局・十四主星を三合派基本法で配置。命盤グリッド表示 |
| インド占星術 | インド | ラヒリ・アヤナーンシャによる恒星黄経。月のナクシャトラ+パダ、九惑星(平均交点ラーフ/ケートゥ含む)、ラグナ、ヴィムショッタリ・ダシャー(120年法) |

検証アンカー(追加分):2012-12-21=13.0.0.0.0・4アハウ・3カンキン(ロングカウント+ハアブ)、紫微斗数の命盤14星配置を手計算と照合、ジョーティシュのアヤナーンシャ約24度・ダシャー期間計算を照合。

## 精度に関する設計判断
- 旧暦は**日本標準時基準の天保暦法**。中国暦と閏月がずれる年がある(2012年・2017年など)ことをサイト内に明記
- 宿曜は「暦法(伝統)」を採用。「天文法(月の実位置)」との流派差も図鑑で説明
- マヤ暦は日本で流行しているドリームスペルではなく**伝統マヤ(考古学標準)**。この違いも図鑑で説明
- 検証済みアンカー:立春2021=2/3 23:59 JST、春節2000/2024/2025/2026、閏4月2020、日柱2000-01-01=戊午、KIN160=2012-12-21(4アハウ)、1995年の外惑星星座など

## 技術構成
- 静的サイト(HTML/CSS/vanilla JS)。ビルド不要、`serve.py`で開発サーバー
- 天文計算:`js/vendor/astronomy.browser.min.js`(astronomy-engine, MIT)
- 暦計算コア:`js/almanac.js`(節気・干支・旧暦・真太陽時・ASC)
- 共通UI:`js/core.js`(プロフィールのlocalStorage保存、入力フォーム、タブ)
- デザイン:深緑の闇に翡翠(緑)・瑠璃(水色)・蓮(ピンク)の仏教チック。明朝体(Shippori Mincho)、曼荼羅光背、蓮弁SVG

## セキュリティ(2026-07-07 監査済み)
- ユーザー入力(名前)は `Uranai.esc()` でHTMLエスケープしてから描画(格納型XSS対策。攻撃入力での実地テスト済み)
- 全ページにCSPメタタグ(script/styleはself+Google Fontsのみ、object-src none)とreferrer no-referrer
- 外部リソースはGoogle Fontsのみ。天文計算ライブラリはローカル同梱。クッキー・トラッキング・秘密情報なし
- 開発サーバー(serve.py)は127.0.0.1バインドでLAN非公開
- **一般公開時のTODO**: HTTPS必須、CSPをHTTPヘッダーへ移行、frame-ancestors(クリックジャッキング対策)はヘッダーでのみ設定可能なのでホスティング側で設定
